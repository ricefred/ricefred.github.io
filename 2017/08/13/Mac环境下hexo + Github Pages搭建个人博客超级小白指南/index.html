安装Hexo

使用Terminal安装。# 后面的内容是评论，直接复制粘贴的话请不要复制#后面的语句。

Hexo常用命令：

  1. hexo generate (hexo g)：生成静态网页，会在当前目录中生成一个叫做public的文件夹，而这个文件夹里就包含了所有静态网页的资源。
  2. hexo server (hexo s)：启动本地web服务，可以用于预览静态网页。
  3. hexo deploy (hexo d)：部署网页到远端（本文部署到Github pages）。
  4. hexo d -g：生成静态网页后部署，属于组合命令。

Github Pages配置

什么是Github Pages

Github Pages本意是用于介绍托管在Github上的项目，不过Github也鼓励使用者用于各种用途，原话就是Whatever you want. 因为其空间免费且稳定，用来放置静态网页的博客是再合适不过了。

如何搭建一个独立博客——简明Github Pages与Hexo教程

值得注意的是，这篇教程中寻找SSH Key的部分是在windows中寻找的。对于Mac OS系统而言，SSH Key会存放在.ssh文件夹中，而这个文件夹是隐藏的。大家可以通过以下操作步骤找到这个文件夹。

  1. 打开Finder后点击Go, 在下拉菜单中选择Go to folder…
  2. 填写~/
  3. 在这个用户文件夹中查看隐藏文件：快捷键是command+shift+.
  4. 找到.ssh文件夹
